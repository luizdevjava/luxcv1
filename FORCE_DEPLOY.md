# FORÇAR DEPLOY MANUAL

## Opção 1: Via GitHub Desktop
1. Abra GitHub Desktop
2. Clone seu repositório
3. Arraste os arquivos do projeto para a pasta
4. Commit: "Force update APIs"
5. Push

## Opção 2: Via Web GitHub
1. Vá para github.com/SEU-USUARIO/lux-companions
2. Clique "Add file" → "Upload files"
3. Arraste a pasta src/app/api
4. Commit changes

## Opção 3: Verificar Vercel
1. Vá para vercel.com/seu-projeto
2. Settings → Git
3. Verifique se está conectado ao repo correto

## Após upload:
1. Vá para Vercel
2. Deployments → Redeploy
3. Aguarde 2-3 minutos
4. Teste: https://lux-companions.vercel.app/api/debug
